How to resize images in Photoshop CS6

  1. Open the image in photoshop.




  2. In the menu bar select "image and then select "image size" option.

  3. In the "image size" window adjust the width of the image inside the "Pixel Dimension" section. The height gets adjusted automatically.



  4. In the menu bar select "File" and then select "Save for web".

  5. A new window will open with 4 tabs. Select the "2-Up" tab which shows the original image and the image optimized for web. Select the image format and adjust the quality of the image. Then click "Save".

  6. In the new window rename the image to whatever you like and then click "Save". Finally you are done.

How to resize images in Gimp

  1. Open the image in Gimp.



  2. In the menu bar select "image" and then select "scale image" option.

  3. In the "scale image" window adjust the width of the image. The height will automatically adjust.



  4. In the menu bar select "File" and then select "Export as".

  5. In the new window choose the image format and rename the image to whatever you desire and click "Export".

  6. A new window will open. Adjust the quality of the image and click "Export".
